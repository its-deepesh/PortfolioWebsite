# 🌐 Portfolio Website

A simple and responsive personal portfolio website built using **HTML5** and **CSS3**. It features a modern navigation bar, a clean hero section, smooth hover effects, and basic animations.

---

## 📌 Features

- 🎨 Modern responsive navigation bar
- 👤 Hero section with introduction
- ✨ Smooth hover effects
- 🎬 Fade-in animations
- 📱 Responsive layout
- 🎯 Clean and minimal design

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

```
Portfolio-Website/

```

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/its-deepesh/Portfolio-Website.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
